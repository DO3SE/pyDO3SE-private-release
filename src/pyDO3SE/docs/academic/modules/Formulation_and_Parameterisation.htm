Model Data

Model data is split into the following categories:

Configuration

This is derived from the configuration file provided by the user.

 

External State

This is derived from the data provided by the user. The data is cleaned and processed ready for use by the DO3SE model (See Input requirements below).

 

Model State

Model state is the internal state of the model. It defines the current state of growth, photosynthesis etc. The output is derived from the model state.
