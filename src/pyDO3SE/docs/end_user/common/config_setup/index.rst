==================
DO3SE Config Setup
==================

Documentation on common model config setup types.

Setting up Config Files
=======================

The shape of the config files is set in :mod:`pyDO3SE.Config.Config_Shape`.

You can generate a config file with :func:`pyDO3SE.tools.cli.config.generate` .
